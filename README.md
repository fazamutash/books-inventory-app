# Books Inventory API

## Installation

```bash
$ npm install
```

## Running the app on Local

```bash
# Running development for first time
$ cp .env.local .env

# development
$ npm run start
```

## Prisma

```bash
# generate Prisma Client
$ npm run prisma:build

# run Prisma Migration & generate Prisma Client
$ npm run prisma:migrate

# run Prisma seed
$ npm run prisma:seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
